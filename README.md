# API Integration Assignment

##  Project Description
This project demonstrates the integration of multiple AI APIs using Python. 
The programs use APIs such as Groq,Cohere,HuggingFace and Ollama to perform tasks like text generation and processing. 
Each API is tested using separate Python scripts

##  Setup Instructions
1. Install Python (version 3.8 or above).
2. Install required packages:
```bash
pip install -r requirements.txt
'''
## Obtaining API Keys
## Groq
Go to https://console.groq.com
Signup and generate API key
## Cohere
Go to https://cohere.ai
Sign up and generate API key
## HuggingFace
Go to https://huggingface.co
Go to settings->Access Tokens Create a new Access Token
## Ollama
Install Ollama locally from https://ollama.ai

## how to run Programs
Run each file using:
'''bash
python groq_example.py
python cohere_test.py
python huggingface_api.py
python ollama_api.py
'''
